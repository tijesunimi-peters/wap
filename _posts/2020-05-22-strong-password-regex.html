Summary: Match strong password during signup/registration

Regular Expression

      (?=.*[A-Z]{1,})(?=.*\d{1,})(?=.*[@#$&_\-*]).{8,}

Parts

  • Look forward for minimum of 1 UPPERCASE text
    {% highlight html %} (?=.*[A-Z]{1,}) {% endhighlight %}

  • Look forward for minimum of 1 DIGIT[0-9] text
    {% highlight html %} (?=.*\d{1,}) {% endhighlight %}

  • Look forward for minimum of 1 SPECIAL CHARACTER
    {% highlight html %} (?=.*[@#$&_\-*]) {% endhighlight %}

  • 8 minimum length
    {% highlight html %} .{8,} {% endhighlight %}
